# Kavin R Premium Full Stack Portfolio

This is a premium portfolio converted into a full stack project.

## Folder Structure

- `frontend/` - React + Vite + Tailwind + Framer Motion portfolio
- `backend/` - Node.js + Express API
- `frontend/src/App.backup.jsx` - Safety backup copy of the current portfolio UI

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Resume Download Setup

Put your resume PDF inside:

```bash
frontend/public/resume.pdf
```

Then the Download Resume button will work.

## API Routes

- `GET /api/profile`
- `GET /api/projects`
- `POST /api/contact`
